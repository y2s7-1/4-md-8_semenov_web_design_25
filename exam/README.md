# MIUZ Diamonds - Ювелирный интернет-магазин

## О проекте
Полнофункциональный интернет-магазин ювелирных украшений премиум-класса. Проект создан для защиты учебного проекта по веб-разработке.

## Технологии
- HTML5 (семантическая верстка)
- CSS3 (Flexbox, Grid, анимации)
- JavaScript (ES6+)
- LocalStorage для корзины
- Адаптивный дизайн (mobile-first)

## Функционал
- Просмотр каталога товаров
- Фильтрация по категориям
- Корзина покупок
- Модальные окна
- Адаптивная верстка
- Форма подписки
- Имитация покупки
